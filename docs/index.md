# Symposion

**Symposion** is a collaborative, role-based multi-agent framework where specialized agents coordinate through explicit messages to complete goals end-to-end.

- [Architecture](./architecture)
- [Roadmap](./roadmap)

## Agent Pipeline

Herm → Daed → Athen → Met → Heph → Nem → Clio

![Symposion agent flow architecture](./assets/symposion-agent-flow-architecture.png)

## Why Symposion?

- Role-specialized agents instead of one monolithic assistant.
- Message-driven orchestration with transparent, replayable coordination.
- Quality gates and revision loops before final outputs are delivered.
